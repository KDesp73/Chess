# Chess

Two player Chess written in C++ with CLI

## Implementations

- Checkmate
- Stalemate
- King and Queen side castling
- En Passant
- Check
- Pins
- Pseudo-valid move checks
- Import - Export FEN position strings
- 3-fold repetition draw
- Draw due to insufficient material
- 50 move rule
- Pawn promotions
- Material advantage

## Limitations

- No computer opponent
- CLI only (no GUI)
